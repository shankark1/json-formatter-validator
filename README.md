JSON Formatter & Validator

A beginner-friendly Python project that validates JSON input and formats it into a clean, readable structure. It also reports helpful error messages when the JSON is invalid.

Features

- ✅ Validate JSON syntax
- ✅ Pretty-print JSON with indentation
- ✅ Display detailed parsing errors
- ✅ Simple command-line interface
- ✅ Lightweight and beginner-friendly

Requirements

- Python 3.8 or later

Run

python main.py
